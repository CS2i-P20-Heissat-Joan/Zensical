# Syntaxe de base

## Gras

Le **gras** s'obtient en entourant le texte par 2 astérisques.

=== "Exemple"
    ```markdown
    Cette phrase est normale.
    **Cette phrase est en gras.**
    Le mot **Markdown** seul est en gras.
    ```

=== "Résultat"
    Cette phrase est normale.
    **Cette phrase est en gras.**
    Le mot **Markdown** seul est en gras.

## Italique

L'*italique* s'obtient en entourant le texte par 1 astérisque, ou 1 souligné (*underscore*).

=== "Exemple"
    ```markdown
    Cette phrase est normale.
    *Cette phrase est en italique.*
    Le mot *Markdown* seul est en italique.
    On peut combiner le **gras *et l'italique***.
    ```

=== "Résultat"
    Cette phrase est normale.
    *Cette phrase est en italique.*
    Le mot *Markdown* seul est en italique.
    On peut combiner le **gras *et l'italique***.

## Texte barré

Un texte est barré s'il est entouré d'un double tilde, soit `~~`.

=== "Exemple"
    ```markdown
    ~~Cette phrase est barrée.~~
    ```

=== "Résultat"
    ~~Cette phrase est barrée.~~

## Surbrillance

Une ==surbrillance== s'obtient en entourant le texte par 2 signes `=`.

=== "Exemple"
    ```markdown
    Cette phrase est normale.
    ==Cette phrase est en surbrillance.==
    Le mot ==Markdown== seul est en surbrillance.
    ```

=== "Résultat"
    Cette phrase est normale.
    ==Cette phrase est en surbrillance.==
    Le mot ==Markdown== seul est en surbrillance.
